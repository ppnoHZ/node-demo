# 小技巧
使用 supervisor可以在代码改变的时候，自动重启脚本。
* 安装  npm install -g supervisor
* 使用  supervisor app.js

# 命令

## 创建express项目

* 3X 的版本：express -t ejs app 创建模板为ejs的项目,默认的为jade模板
* 4X 的版本：express -e ejs app 创建模板为ejs的项目,默认的为jade模板
